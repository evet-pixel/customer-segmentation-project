# Customer Segmentation using K-Means Clustering

## Objective
Segment customers based on income and spending behavior using K-Means clustering.

## Tools Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Excel

## Dataset
Customer dataset containing:
- CustomerID
- Age
- Income
- SpendingScore

## Methodology
1. Load dataset using Pandas.
2. Apply K-Means clustering.
3. Generate customer segments.
4. Visualize clusters using a scatter plot.
5. Export results to Excel.

## Output
Customers are grouped into 3 clusters based on spending patterns and income.
